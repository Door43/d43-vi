===== Setting Up a Word Processor =====

This module answers the question: How do I set up a Word Processor?\\
Before you start this module have you learned about:
  * [[vi:ta:vol1:process:platforms|Selecting a Platform]]

==== Obtaining the Source Text for Open Bible Stories ====

  - Go to [[:obs-template|Get Open Bible Stories Word Template]]
  - Select your source language (the language from which you will translate) and click "Save". (Note: If you click the "Add all images in the file", the download size will increase greatly.) **The template will download as a docx file.**
  - Open this template in your word processor of choice.
