===== Translation Notes: =====

The list of names of men and the places they came from continues.

    * **Shammah, Hararite**  - Translate the same way you did in [[vi:bible:notes:2sa:23:11]].
    * **Ahiam, Sharar, Ahasbai, Hezro, Paarai, Igal, Bani**  - Names of men (See: [[vi:ta:vol1:translate:translate_names]]).
    * **Eliphelet**  - Translate the same way you did in [[vi:bible:notes:2sa:05:13]].
    * **Maacathite**  - Someone decended from Maacah which you translated in [[vi:bible:notes:2sa:03:02]].
    * **Eliam**  - Translate the same way you did in [[vi:bible:notes:2sa:11:02]].
    * **Ahithophel, Gilonite**  - Translate name the same way you did in [[vi:bible:notes:2sa:15:11]]. The name of the place here has an ending, "-ite" which means someone from that place.
    * **Carmelite**  - The name of the place here has an ending, "-ite" which means someone from the same place you translated in [[vi:bible:notes:2sa:02:01]].
    * **Zobah**  - Translate the same way you did in [[vi:bible:notes:2sa:08:03]].
